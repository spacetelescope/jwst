Reference Selection Keywords for THROUGHPUT
+++++++++++++++++++++++++++++++++++++++++++
CRDS selects appropriate THROUGHPUT references based on the following keywords.
THROUGHPUT is not applicable for instruments not in the table.
All keywords used for file selection are *required*.

========== ================================================
Instrument Keywords                                         
========== ================================================
NIRISS     INSTRUME, FILTER, DATE-OBS, TIME-OBS
