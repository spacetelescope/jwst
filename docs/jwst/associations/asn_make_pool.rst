asn_make_pool
=============

Association pool creation from a list of FITS files can be done
either using the command line tool ``asn_make_pool`` or through the Python API
:py:func:`~jwst.associations.mkpool`.

Command Line
------------

.. code-block:: shell

   asn_make_pool --help
